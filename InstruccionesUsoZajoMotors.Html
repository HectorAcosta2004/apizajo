Instrucciones de Uso

Última actualización: 12 de mayo de 2026

1. Introducción

Bienvenido a Zajo Motors. Estas instrucciones tienen como finalidad orientar al usuario sobre el correcto uso de la aplicación, sitio web y servicios digitales ofrecidos.

2. Requisitos para el uso

  • Contar con conexión a internet estable.
  • Utilizar un dispositivo compatible.
  • Mantener actualizada la aplicación o navegador.
  • Proporcionar información verídica al registrarse.

3. Registro y acceso

Para acceder a ciertas funciones, el usuario deberá crear una cuenta proporcionando datos válidos y actualizados.

  • No compartir credenciales de acceso.
  • Mantener la confidencialidad de la contraseña.
  • Notificar cualquier acceso no autorizado.

4. Uso correcto de la plataforma

El usuario se compromete a:

  • Utilizar la plataforma únicamente para fines legales.
  • No alterar ni dañar el sistema.
  • No intentar acceder a información restringida.
  • No realizar actividades fraudulentas.

5. Servicios disponibles

La plataforma puede incluir funcionalidades relacionadas con:

  • Consulta de vehículos.
  • Agendamiento de servicios.
  • Seguimiento de mantenimiento.
  • Atención y soporte al cliente.
  • Información automotriz.

6. Recomendaciones de seguridad

Importante: Nunca compartas información confidencial con terceros y evita utilizar redes públicas inseguras para acceder a tu cuenta.
  • Cerrar sesión después de usar la plataforma.
  • Utilizar contraseñas seguras.
  • Mantener actualizado el dispositivo.
  • Reportar errores o actividades sospechosas.

7. Limitación de responsabilidad

Zajo Motors no se hace responsable por fallos ocasionados por problemas externos, interrupciones del servicio de internet, uso indebido de la plataforma o daños ocasionados por terceros.

8. Actualizaciones

Estas instrucciones pueden modificarse en cualquier momento para mejorar la experiencia del usuario o adaptarse a nuevas funcionalidades.
